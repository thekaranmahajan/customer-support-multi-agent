# Demo Script

1. Start Ollama and confirm `llama3.2` and `nomic-embed-text` are available.
2. Start the FastAPI backend.
3. Start the Streamlit UI.
4. Upload a sample PDF policy document from the sidebar.
5. Ask: `What is the current refund policy?`
6. Ask: `Give me a quick overview of customer Ema's profile and past support ticket details.`
7. Ask: `Summarize Raj's open tickets and the refund policy in one response.`
8. Show the MCP server command and explain that the same backend logic is exposed as MCP tools.
